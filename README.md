# Прошивка 2D Манипулятора на базе ESP32-S3

Настоящее микропрограммное обеспечение предназначено для управления двухкоординатным манипулятором, оснащенным шаговыми двигателями, электромагнитным захватом и модулем радиочастотной идентификации (MFRC522). Программный код реализован на языке C++ с использованием фреймворка Arduino и среды разработки PlatformIO. Для обеспечения многозадачности применяется операционная система реального времени FreeRTOS.

## Основные функциональные возможности

*   **Асинхронное управление электроприводами**: Применение библиотеки `FastAccelStepper` позволяет осуществлять прецизионное управление шаговыми двигателями с реализацией линейной интерполяции и расчетом векторного движения.
*   **Автоматическая калибровка (Homing)**: Алгоритм возврата в исходное положение включает программную фильтрацию дребезга контактов, защиту от электромагнитных помех и предотвращение механических перегрузок.
*   **Интерфейс связи по протоколу UART**: Обеспечивается двунаправленный обмен данными с управляющим сервером (например, на базе Node.js или одноплатного компьютера).
*   **Интеграция RFID-технологии**: Реализовано чтение уникальных идентификаторов (UID) радиочастотных меток в режиме реального времени для задач позиционирования и идентификации объектов.
*   **Управление электромагнитным захватом**: Обеспечивается фиксация и удержание перемещаемых грузов.

## Протокол информационного обмена (UART API)

Взаимодействие микроконтроллера с управляющим устройством осуществляется посредством текстовых команд (параметры интерфейса: 115200 бод, 8 бит данных, без контроля четности, 1 стоп-бит). Каждая управляющая последовательность должна завершаться символом перевода строки (`\n`).

### Спецификация команд:

*   **`HOME`**
    *   **Описание:** Инициализирует процедуру калибровки (поиск нулевой координаты по обеим осям).
    *   **Ответ системы:** `HOME_OK` по завершении процедуры.
*   **`M <targetX> <targetY>`**
    *   **Описание:** Перемещение рабочего органа в заданные координаты X и Y с использованием алгоритма линейной интерполяции.
    *   **Пример вызова:** `M 1200 4500`
    *   **Ответ системы:** `OK` при достижении целевых координат.
*   **`MAG <state>`**
    *   **Описание:** Управление состоянием электромагнитного захвата. Значение `1` соответствует включению, `0` — отключению.
    *   **Пример вызова:** `MAG 1`
    *   **Ответ системы:** `OK`
*   **`SCAN`**
    *   **Описание:** Выполняет опрос RFID-считывателя на наличие метки в рабочей зоне.
    *   **Ответ системы:** `RES UID <UID_метки>` в случае успешного считывания, либо `RES UID NONE` при отсутствии метки.

## Инструкция по сборке и развертыванию (PlatformIO)

Проект настроен для использования в среде PlatformIO. Разрешение зависимостей и загрузка библиотек производятся автоматически.

### Программные зависимости (определены в `platformio.ini`):
*   `miguelbalboa/MFRC522` (версия 1.4.11 или новее)
*   `gin66/FastAccelStepper` (версия 0.30.11 или новее)

### Порядок сборки:

1.  Установите интегрированную среду разработки [Visual Studio Code](https://code.visualstudio.com/) с расширением [PlatformIO IDE](https://platformio.org/).
2.  Клонируйте репозиторий и откройте директорию проекта в Visual Studio Code.
3.  Осуществите подключение отладочной платы ESP32-S3 DevKitC-1 к персональному компьютеру посредством интерфейса USB.
4.  Инициируйте процесс компиляции нажатием кнопки **Build** (✓), расположенной на нижней панели PlatformIO.
5.  Выполните загрузку микропрограммного обеспечения, используя кнопку **Upload** (→).
